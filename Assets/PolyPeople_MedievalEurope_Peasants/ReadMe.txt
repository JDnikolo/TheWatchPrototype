■ Package Contents

- Meshes: 8 FBX files  
(Adult, Elder, Kid, YoungAdult / Female, Male).

- Materials: 4 total
  - Standard: 2 (ColorA, ColorB)  
  - Shader Graph: 2 (ColorA, ColorB) – customizable skin color

- Prefabs: 32 total
  - 4 prefab variants per character (based on material)
  - All configured with Humanoid rig

- Textures:
  - 2 base color textures (512x512)

- Scenes:
  - 1 demo scene showcasing all prefabs

---

■ Geometry

Character mesh triangle count: 456–608 tris. 

---

■ Customizing Skin Color (for CustomSkin Materials)

You can customize the skin color by adjusting the `Color` input node in the Shader Graph used by the material.
